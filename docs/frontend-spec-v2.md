# 前端重构 Spec v2 (极简美学版)

## 1. 核心设计理念
- **完全自包含**：0 外部依赖，所有 CSS/JS 内联，确保在离线/内网环境 100% 样式还原。
- **现代暗黑风**：背景采用深邃蓝黑 (#0a0a14)，卡片采用半透明磨砂质感 (#16162a)。
- **动效反馈**：所有按钮带微交互、加载中带动画、操作成功带 Toast 提示。

## 2. 视觉规范
- **色彩**：
  - Primary: `linear-gradient(135deg, #6366f1 0%, #8b5cf6 100%)` (靛蓝-紫渐变)
  - Success: `#10b981` (翠绿)
  - Danger: `#f43f5e` (玫瑰红)
  - Warning: `#f59e0b` (琥珀橙)
  - Info: `#3b82f6` (天蓝)
- **字体**：优先使用系统等宽字体 (JetBrains Mono / Cascadia Code) 处理日志，UI 使用 Inter/System-UI。
- **阴影**：使用多层级柔和阴影，增加空间感。

## 3. 页面布局
- **登录页**：
  - 全屏背景渐变动效。
  - 居中登录卡片：高斯模糊背景，输入框带 Focus 环绕发光。
- **主面板 (Sidebar + Content)**：
  - **左侧侧边栏**：图标 + 文字，精简导航（注册、结果、设置）。
  - **顶部状态栏**：显示当前用户、服务健康状态、退出按钮。
  - **底部固定日志区**：高度 240px，独立滚动，支持一键清空、折叠。
- **内容区面板**：
  - **注册面板**：大卡片设计，分平台 Tab，参数输入清晰明了。
  - **结果面板**：数据表格（平台、邮箱、凭据预览、时间），每行右侧「操作」组：复制 JSON、查看详情、下载单个。
  - **设置面板**：分组卡片，支持 AHEM、YYDS、代理池、密码修改。

## 4. 关键功能增强
- **实时日志流**：不同颜色的前缀（[+] 绿, [-] 红, [*] 蓝, [!] 黄），自动追随到底部。
- **凭据展示**：在结果表格中点击「查看」弹出半透明模态框，格式化展示完整 JSON 凭据。
- **持久化与导出**：
  - 注册结果自动存入 LocalStorage，刷新不丢。
  - 顶部按钮支持「导出全部 JSON」和「导出全部 TXT」。
- **修改账号**：修改用户名或密码后，自动清理 Token 并弹窗提示重新登录。

## 5. 技术实现
- **CSS**：使用 CSS Variables 管理主题，Flexbox/Grid 响应式布局。
- **JS**：封装 `api` 调用工具、`SSE` 流处理器、`toast` 通知系统。
- **安全性**：敏感配置（如 YYDS Key）在展示时自动打码。
